- This is java code for the proof of concept "Spectral Clustering of Attributed Graphs", the idea is described in the draft spectralMDLMain.pdf

- the paper is available in the svn under https://svn01.cs.univie.ac.at/dm-projects/forschung/attributedSpectral/tex

-All relevant code is in the package attributedEmbedding

- The main Method is MixedSpectralMain

- the folder highSchool contains data from 
Rossana Mastrandrea, Julie Fournet, and Alain Barrat. 2015. Contact Patterns in
a High School: A Comparison between Data Collected Using Wearable Sensors,
Contact Diaries and Friendship Surveys.
